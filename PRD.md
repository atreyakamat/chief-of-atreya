# PRD: Personal Workforce Agentic Servicing (ZEN OS)

## 1. Vision & Executive Summary
**Personal Workforce Agentic Servicing (ZEN OS)** is a next-generation AI operating system designed to act as a fully autonomous executive assistant. Unlike standard chat-bots, ZEN OS integrates deeply with the user's computer, browser, and cloud services to proactively manage work, siloing projects into intelligent "Workspaces," managing schedules via a "Neural Calendar," and serving as a high-speed "Neural Link Manager."

## 2. Core Features (Implemented & Proposed)

### 📂 Intelligent Workspace Servicing
*   **Context Siloing:** Every project (e.g., Pixel N Purpose) has its own memory and task set.
*   **Autonomous Orchestration:** Zen can open apps, type, and click to complete tasks within a workspace.
*   **Auto-Meeting Generation:** Ability to instantly create and host Google Meet sessions for workspace collaboration.

### 💼 Spark+ Business Servicing (Partnership Engine)
*   **Lead Management:** Integrated CRM for Colleges, Employers, and Project Partners.
*   **Metric Tracking:** Automated tracking of success criteria (Onboarded Employers, Pathway Count, External Projects).
*   **Autonomous Follow-ups:** AI handles demo scheduling and persistence for partnership growth.
*   **Skill Pathway Coordination:** Tools to validate industry requirements and sync them with learner outcomes.

### 📅 Neural Calendar & Time Tracking
*   **Unified Timeline:** Syncs Google Calendar with local neural events.
*   **OCR Event Extraction:** Zen "sees" a schedule on screen (via Alt+S) and populates the calendar automatically.
*   **Work Tracking:** Clock-in/Clock-out system to track deep work sessions.

### 🔗 Neural Link Manager & Browser Extension
*   **URL Ingestion:** Capture links from any source (Chat, Mobile Ingest, or Browser).
*   **AI Summarization:** Every saved link is automatically analyzed and summarized by Zen.
*   **Chrome Extension:** A dedicated browser companion that "sends" current tabs to ZEN OS for long-term storage and servicing.

### 🤖 Autonomous "Jarvis" Interface
*   **Hey Zen Wake Word:** Multi-modal activation (Voice + Hotkey).
*   **Zen Activation Popup:** Immediate Windows-native visual feedback for all AI actions.
*   **Auto-Healing Engine:** System-level error correction using Gemini to fix PowerShell/System commands.

## 3. Technical Stack
*   **Core:** Electron (Node.js)
*   **Frontend:** Vanilla JS / CSS (Cyber-Luxe & Neural-Light Themes)
*   **Brain:** Multi-Model Gateway (Groq, OpenRouter/Gemini, NVIDIA, Ollama)
*   **Database:** SQLite (Better-SQLite3)
*   **Vision:** Tesseract OCR / AI Vision (Llava/Gemini)
*   **Voice:** Windows Native TTS / SpeechRecognition

## 4. Success Metrics
*   **Latency:** Neural responses < 1.5s (Cloud) / < 3s (Local).
*   **Autonomy:** 90% success rate in Auto-Healing system commands.
*   **Utility:** Reduction in manual context-switching via Workspace Silos.
